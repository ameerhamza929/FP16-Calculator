About FP16 Floating Point

FP16 (16-bit floating point) is a binary floating-point format that occupies 16 bits (2 bytes) in computer memory.

Key Features:

  • 5 bits exponent (with bias of 15)
  • 10 bits significand (with implicit leading bit)
  • Range: ±65,504
  • Precision: ~3 decimal digits
  • Special values: ±Infinity, NaN, ±0

Common Uses:

  • Machine learning (especially neural networks)
  • Graphics processing
  • Applications where memory bandwidth is critical

Input Formats:

You can enter numbers in either decimal (e.g., 3.14) or hexadecimal FP16 format (e.g., 0x4248).

Supported Operations:

  • Addition (+)
  • Subtraction (−)
  • Multiplication (×)
  • Division (÷)

About Precision:

FP16 has limited precision compared to 32-bit or 64-bit floating point. Some operations may show rounding errors due to this limited precision.
